SCREENSHOTS

When I looked for an app that can support Markdown and save my post online with my Android phone, I found nothing. So, I try to make one application to match my need. That's why I develop Muggle.
It's developed base on another open source app named "Mua", which can only edit a file offline. I got the source code of Mua and add some new module into it, in this way, we can now create Markdown file with Github.com to save file online. It has been named as "Muggle" which means "Mug" + "Github", and also Muggle in Harry Potter means those people who have no magical power like us.
I hope the Muggle App can give us magical power and easy for everyone to use. Just enjoy it!
